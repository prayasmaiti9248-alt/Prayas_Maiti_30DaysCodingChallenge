# Day 11 - Tree Level Order Traversal

## Approach (Recursive)

1. Find height of tree
2. For each level:
   - Print nodes at that level

## Key Idea:
Traverse tree level by level using recursion.

## Alternative (Optimal):
Use queue (BFS) for O(n) traversal.

## Complexity:
- Recursive: O(n^2)
- BFS: O(n)
