# Simple Array Sum

## Problem
Given an array of integers, find the sum of its elements.

---

## Approach

### Intuition:
To compute the total sum, iterate through the array and keep adding each element to a running sum.

### Steps:
1. Initialize sum = 0
2. Traverse the array
3. Add each element to sum
4. Return sum

---

## Complexity
- **Time:** O(n)  
- **Space:** O(1)

---

## Key Points
- Basic iteration problem  
- Helps build foundation for array traversal  
- Important for understanding prefix sums later
